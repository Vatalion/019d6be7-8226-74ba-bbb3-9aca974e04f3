# 6. Progressive decentralization

**Canonical:** [prd.md](../../_bmad-output/planning-artifacts/prd.md) §6

## Phase 1 — Coordinated marketplace (CURRENT)

- Wallet-to-wallet USDT/USDC; platform **does not hold funds**
- Trade deadlines, chat, moderator disputes, reputation
- **Say:** “Crypto classifieds with platform-backed trade rules”
- **Do not say:** “trustless escrow”, “atomic swap”

## Phase 2 — Verified + fulfillment

- Explorer verification for active networks
- Proven pickup or one carrier; no mock shipping in prod

## Phase 3 — Trustless settlement

- ICRC / canister escrow
- Optional HTLC; jury only if moderation fails to scale
