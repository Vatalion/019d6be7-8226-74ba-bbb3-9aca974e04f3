# 1. Executive summary

**Canonical:** [prd.md](../../_bmad-output/planning-artifacts/prd.md)

CryptoMarket P2P is an **OLX-style classifieds marketplace** on the Internet Computer where users buy and sell **goods** using **USDT/USDC**. Users pay **directly wallet-to-wallet**; the platform enforces trade structure, deadlines, messaging, and disputes without converting crypto to fiat.

**Differentiators:**

- Goods-first commerce (not a DEX)
- Stablecoins only — no fiat layer on platform
- Pseudonymous Internet Identity
- Phased path to verified payments and on-chain escrow
