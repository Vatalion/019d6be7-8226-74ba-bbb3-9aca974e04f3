# 0. MVP scope

**Canonical:** [prd.md](../../_bmad-output/planning-artifacts/prd.md) · [product-brief.md](../../_bmad-output/planning-artifacts/product-brief.md)

## In scope (Phase 1)

- Web app on Caffeine (React + Motoko)
- Internet Identity; pseudonymous profiles
- Listings (physical pickup + digital)
- Trades with **wallet-to-wallet** USDT/USDC (4 networks)
- Platform state machine, chat, moderator disputes, reputation
- Admin + observability

## Out of scope (Phase 1)

- Fiat rails, bank/card payments
- On-chain fund custody / HTLC
- Mandatory KYC
- Governance/vault/jury as launch features
- Multi-carrier shipping in UI (pickup only)
